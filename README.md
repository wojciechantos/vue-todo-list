## Backend development
```
Download mongoDB image for docker and run (for example with Docker Desktop)

cd ./backend
npm install
node server.js
```

## Frontend development
```
cd ./backend
npm install
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
